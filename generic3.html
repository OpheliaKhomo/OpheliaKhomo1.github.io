History of Nobel Prize Winners Analysis

Project Overview

The project involves analyzing the SAT performance of public schools in New York City using the schools dataset which measures students' literacy, numeracy, and writing skills. The test consists of three sections: Reading, Math, and Writing, each with a maximum score of 800 points

Methodology

  • Data Exploration and cleaning for missing values and inconsistencies
  • Performed statistical analysis to describe the performance distribution across schools

Tools utilized

  • Pandas library for data manipulation & statistical analysis
  • Numpy for numerical computations
  • Matplotlib & Seaborn fpr data visualization
